This repo shows how we start working with mocking in python 3.6.6

# prerequisites
install pyenv to install 3.6 and then pipenv ref. bit.ly/nnpipenv

# get started

```bash

cd "$CODE"
    # install pip packages
    pipenv sync

    # run test 
    pipenv run  pytest
```
